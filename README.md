# Project Documentation

## Introduction
This repository contains the project documentation for the 3D Learning Application.

## Features
- Interactive 3D model display
- Educational resources
- User-friendly interface

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Nam-Anh1806/h-nh-h-c-3D.git
   ```
2. Navigate to the project directory:
   ```bash
   cd h-nh-h-c-3D
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage
To launch the application, run:
```bash
npm start
```
